Function lib_L1D_config_SRAM configures L1D to have the desired SRAM size.

Parameters
[in]size_SRAMdesired L1D SRAM size in bytes
Return values
LIB_CACHE_SUCCESSSuccess. No error.
LIB_CACHE_ERRORFailure.
Remarks
L1D will be write-back invalidated before being configured.
To avoid unnecessary cache configuration, users can first call function lib_get_L1D_SRAM_size() to find out current SRAM size. If it is the desired size, this function does not need to be called.
